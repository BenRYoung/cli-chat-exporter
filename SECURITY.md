# Security Policy

## Reporting a Vulnerability

Please report security issues privately through GitHub Security Advisories when available, or by contacting the maintainer through the repository issue tracker with a request for a private contact path.

Do not include private chat histories, tokens, credentials, or full export archives in public issues.

## Local Data Handling

CLI Chat Exporter reads local files and writes local archive files. It does not upload chat records.

Users are responsible for protecting generated exports because detailed exports may contain prompts, tool payloads, local paths, and other sensitive context.
